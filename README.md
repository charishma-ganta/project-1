# Movie Streaming Apps – HTML & CSS Project
This is a beginner-level static website created using **HTML and CSS** ..  
The project demonstrates basic front-end development concepts such as page structure, styling, and layout

##  Project Description
The website displays a list of popular movie and video streaming platforms in a simple and organized layout.  
Each platform is shown inside a card with its name, description, and logo.  
Clicking on a logo redirects the user to the official website of that platform.

Navigation links at the top allow easy scrolling to each section of the page.

##  Technologies Used

- **HTML5** – Used to structure the webpage content  
- **CSS3** – Used for styling, layout, and basic animations  

This project uses **only HTML and CSS** and does not include JavaScript or external frameworks.

##  Features
- Navigation bar with anchor links
- Card-based layout using Flexbox
- Hover effects on cards and images
- Simple responsive design
- Clean and readable code structure
